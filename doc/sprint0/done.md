Definition of Done Checklist:

- Code:  
  - MVC connectivity is functional  
  - All implemented features are completed and correctly functional  
  - no known bugs or unfinished features  
  - Code is junit tested  
  - Code is peer reviewed  
  - UI is up to group member standards

- Documentation:  
  - Documents fit all mentioned requirements  
  - Documents are peer reviewed  
  - Resources are credited

- User stories:   
  - User Stories shows what tasks the Users want to be able to complete with the app and the app has the ability to complete those tasks  
    User Stories fits all the requirements and is relevant to the application  
  - User stories are peer reviewed  
  - Resources are credited
